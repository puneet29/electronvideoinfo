# Electron App Beginner

## Getting Started

1. Install npm (if not):

    ```bash
    sudo apt install npm
    ```

2. Install electron dependency:

    ```bash
    cd videoinfo
    npm init
    npm install --save electron
    sudo apt install ffmpeg
    npm install --save fluent-ffmpeg
    ```

3. Run app:

    ```bash
    npm run electron
    ```
